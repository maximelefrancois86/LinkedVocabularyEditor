You don't find the data property you are looking for?
  • It belongs to a vocabulary that has not been imported yet, hence it has no wiki page:

    Write your need in the moreinfo box, and ask the sysadmin for review.

  • It belongs to the vocabulary you are developing, but it has no wiki page yet:
    1. create it first and declare it as a data property: [create new resource]
    2. re-open this dialog.
  • It has a wiki page, but it is not declared as a data property:

    Make sure you need it to be a data property, then:

    1. declare it as a data property: one that link individuals to simple values;
    2. re-open this dialog.
